# LRTHW

### To get started

* Install ruby.
* Install git if you don't have it installed.
* Then follow the step below:

#### Clone this repo
```bash
git clone https://github.com/krahman/lrthw.git
```

#### Go to lrthw directory
```bash
cd lrthw
```

#### Run excertice (e.g.; ex10)
```bash
ruby ex10.rb
```
